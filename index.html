Вітаємо на WEBSTART.UA!

WEBSTART.UA – це ідеальне місце для тих, хто робить перші кроки у світі веб-розробки. Ми пропонуємо повний спектр послуг для створення та підтримки вашого веб-сайту. Якщо ви новачок у цьому бізнесі, ми допоможемо вам освоїти всі необхідні навички та досягти успіху в онлайні.

Приклад HTML:

index.html

<!DOCTYPE html>
<html lang="uk">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">

</html>

HTML

HTML (HyperText Markup Language) – це стандартна мова розмітки для створення веб-сторінок. Вона використовується для структурування вмісту на веб-сторінках і його відображення в браузері.

CSS

CSS (Cascading Style Sheets) — це мова стилів, яка використовується для опису вигляду та форматування документів, написаних мовою розмітки HTML. CSS відокремлює структуру документа від його стилізації, дозволяючи розробникам ефективно управляти зовнішнім виглядом веб-сторінок.

Приклад CSS:

style.css

body {
    background-color: lightblue; /* Задає колір фону для сторінки */
    color: #333; /* Задає темно-сірий колір тексту для всієї сторінки */
}
  
h1 {
    color: white; /* Задає білий колір тексту заголовку h1 */
    text-align: center; /* Вирівнює текст по центру */
}
  
p {
    font-family: verdana; /* Встановлює шрифт для абзаців */
    font-size: 20px; /* Встановлює розмір тексту для абзаців */
}

Приклад JavaScript:

script.js

JavaScript

JavaScript — це мова програмування, яка використовується для додавання інтерактивності до веб-сторінок. На відміну від HTML і CSS, JavaScript є мовою сценаріїв, яка дозволяє створювати динамічний вміст на веб-сторінках.

PHP

PHP (Hypertext Preprocessor) — це серверна мова програмування, яка використовується для створення динамічних веб-сторінок та веб-додатків. PHP вставляється в HTML-код і виконується на сервері, генеруючи HTML для відправки користувачам.

Приклад PHP:

index.php

<?php
// Вивід тексту
echo "Hello, world!<br>";
    
// Оголошення змінної
$name = "John";
echo "Hello, " . $name . "<br>";
    
// Функція
function add($a, $b) {
    return $a + $b;
}
echo add(5, 3) . "<br>";
    
// Умовний оператор
$age = 20;
if ($age >= 18) {
    echo "Adult<br>";
} else {
    echo "Minor<br>";
}
?>
